File Structure

 
All the file are well organized, its so easy to work with the template.
 
1. Unpack the files.
 
2. Open "Speekloud HTMLs" folder.
 
3. Open one folder from the 3 versions.
 
You will find the the index.html file on the root that will be edited.
 
Also you will see 3 directories, css, js, fonts and img
 
In the first folder Content you will find style.css and other styling files the all the other files expect the htmls.
 
Js folder contain only the jquery library. Fonts folder contain essetial font files for fonts.

Menu

 
To add a new menu item you just need to add a new list element. In order to make the navigation work properly you need to set an anchor to the desired section of the page.
 

	<div class="collapse navbar-collapse">
		<nav>
			<ul class="nav navbar-nav">                                                                                    
				<li class="active"><a href="#home">Home</a></li>
				<li><a href="about-me.html">About me</a></li>
				<li><a href="category.html">Categories</a></li>
				<li><a href="single-posts.html">Single posts</a></li>
				<li><a href="contact.html">Contact</a></li>
			</ul>
		</nav>
	</div><!-- navbar-collapse end -->

Change the images in different category items

 
You can easily change the images in different section as sections have many similarities. Just replace your image with img/category-img-1.jpg tag.
 

		<div class="category-img">
			<img src="img/category-img-1.jpg" alt="">
			<div class="category-overlay">
			</div>
		</div>

Change the images of different ad in category widget section

 
You can easily change the images of different ad by replacing your image with img/category-widget-ad.png tag.
 

    <div class="category-widget-single">
        <div class="category-widget-single-ad">
            <a href=""><img src="img/category-widget-ad.png" alt=""></a>
        </div>
    </div>

Change the images of instagram section

 
You can easily change the images of instagram by changing userId and accessToken. By creating a account in instagram you can get userId and accessToken and also you can maintain number of image by limit in your website.
 

		var userFeed = new Instafeed({
        get: 'user',
        limit: "8",
        userId: 11175379,
        accessToken: '11175379.467ede5.f5ec3337edc941678b80d5eac0310213',
        template: '<a href="{{link}}" target="_blank"><img src="{{image}}" /><div class="instagram-icon"><i class="fa fa-instagram"></i></div></a>'
    });
    userFeed.run();

Change the video and audio link in your blog

 
Replace your expected links embed code in the iframe tag. By using height and width attribute you can maintain video and audio area in your blog.
 

    <div class="video-area-border-content">
		<div class="col-md-6">
			<div class="video-area-content-detail">
				<iframe src="https://www.youtube.com/embed/KGTfD6P7gu8"</iframe>
			</div>
		</div>
	</div>
